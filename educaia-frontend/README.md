# 🎨 EducaIA – Frontend
Interface web do sistema EducaIA, desenvolvida com **React**.

---

## 🛠️ Tecnologias Utilizadas
- React 18
- React Router (navegação entre telas)
- Axios (requisições para a API)
- React Markdown (formatação do conteúdo)
- jsPDF + html2canvas (exportação para PDF)
- CSS (estilização e layout responsivo)

---

## 📂 Estrutura de Pastas
educaia-frontend/
├── public/ # Arquivos públicos (ícones, página inicial)
├── src/
│ ├── pages/ # Telas: Login, Painel Principal
│ ├── styles/ # Arquivos de estilo CSS
│ └── App.jsx # Configuração de rotas
├── package.json # Dependências do projeto
└── README.md # Este arquivo
plaintext

---

## 🚀 Como Executar
### 1. Instalar as dependências

npm install

2. Iniciar o servidor de desenvolvimento
Run
npm start
A interface abrirá automaticamente em: http://localhost:3000

#🔌 Conexão com o Backend
A URL base da API para comunicação é:

http://localhost:5276/api

#✨ Funcionalidades da Interface
Tela de cadastro e login seguro para professores
Painel com o gerador inteligente de planos de aula
Alternância entre visualização de texto simples ou formatado
Exportação direta de materiais em PDF
Lista, edição e exclusão de materiais salvos
Layout adaptável para computadores e tablets
